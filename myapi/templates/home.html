Hello, Developers!

This is a simple REST-API made by using Django-rest-framework, In this api roots you will find the all the details of gun , medicines & equipment of pubg mobile in the form JSON , which can developers use to develop a website which helps a player to analyse the guns & improve their gameplay

It is on the current basis , I will update it if any comes

Check all roots below
How to use api roots?
For routes click on right top button Api root

URls defined as "/api/type" where type belongs from below:-

  • "api/ar" path shows all automatic Rifle details
  • "api/smg" path shows all shot-machine gun details
  • "api/dmr" path shows all marksman Rifle details
  • "api/lmg" path shows all light-machine gun details
  • "api/sniper" path shows all sniper Rifle details
  • "api/pistol" path shows all pistol details
  • "api/shotgun" path shows all shotgun details
  • "api/other" path shows all melee weapon details
  • "api/health" path shows all medicines details
  • "api/grip" path shows all grip and add on details
Details/terms of API
Guntype

All the terms of gun details described below

  • "name" : Gun name
  • "ammo" : ammo type used in game
  • "mag" : rounds per magazine non-extended
  • "ext_mag" : rounds per magazine extended
  • "fire_rate" : seconds between shots
  • "damage" : initial damage to 100 health player
  • "bullet_speed" : bullet speed m/s
  • "rload" : seconds to reload
  • "shot_range" : ReferenceDistance from game files
  • "scoped_spread" : spread * scoped modifier Lower is better, less spread while scoped
  • "dps" : damage / rate
Check API example
Medicines

All the terms of Medicines details described below

  • "name" : Medicine name
  • "health_gained" : Health Gained
  • "action_time" : Action time
  • "boost_bar" : Boost Bar fill
  • "duration" : Duration
Check API
Grips and addons

All the terms of Grips and addons details described below

  • "name" : Grip & addon name
  • "DRP" : DecreaseRecoilPatternScale
  • "H_recoil" : RecoilMultiplierHorizontal
  • "V_recoil" : RecoilMultiplierVertical
  • "sway" : SwayMultiplier
  • "kick_animation" : AnimationKickMultiplier
  • "dec_adsSpeed" : ADSSpeedMultiplier
